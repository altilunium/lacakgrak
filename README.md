# lacakgrak
Inbound/outbound packet sniffer

![image](https://github.com/user-attachments/assets/1c8614bf-b622-48a7-a851-f18dca5e7d5c)

* Statistics will be updated each 10 seconds
* For each target IP, sorted by byte size
